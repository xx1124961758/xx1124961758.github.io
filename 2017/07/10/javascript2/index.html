JavaScript 中的执行上下文（栈）是什么？

发表于 2017-07-10 | 阅读次数

原文

在这篇文章中，我们讨论下 JavaScript 最基础的部分之一 - Excution Context（执行上下文）。读完全文，你将对 解释器 试图做什么，为什么一些函数/变量在声明之前就可以使用，它们的值如何确定的等问题有一个清晰的认识。

什么是执行上下文？

当 JavaScript 的代码运行时，其运行所处的环境非常重要。大概涉及到的有：

  • 全局代码 - 代码第一次运行时默认的环境
  • 函数内部的代码 - 当执行流进去函数体的时候
  • Eval 代码 - 在 eval 函数体内部执行的文本
